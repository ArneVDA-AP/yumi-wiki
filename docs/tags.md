# Tags

Every page in this wiki carries a small set of tags in its frontmatter — its
structural type (`overview` / `reference` / `subsystem` / `guide` / `design`)
plus the topics it touches (`tauri`, `rust`, `react`, `ipc`, `stream-json`,
`mcp`, `thinking-proxy`, `sqlite`, `agents`, `reliability`, `build`, …). The
index below is generated by the MkDocs Material **tags** plugin; use it to jump
across pages that share a subsystem or a concept.
